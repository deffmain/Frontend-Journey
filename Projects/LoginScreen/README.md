## 📌 Sobre o Projeto

Este é um projeto simples de uma **tela de login** desenvolvida com **HTML e CSS**, com foco na prática de conceitos básicos de desenvolvimento web.

---

## 🎯 Objetivo

Praticar:

* Estruturação de páginas com HTML
* Estilização com CSS
* Organização visual de elementos na tela

---

## 🖥️ Funcionalidades

* Formulário de login centralizado
* Imagem de fundo para melhor experiência visual

---

## 🚀 Observações

Este projeto faz parte dos meus estudos iniciais em desenvolvimento web, com foco em evolução contínua e aplicação de boas práticas de design.
